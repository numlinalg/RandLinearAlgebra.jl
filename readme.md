RandLinearAlgebra is a Julia package that implements standard Randomized Linear Algebra algorithms and provides means for performance comparison.

## Installation
This package is regularly registered with Julia's general registry. To install it, run 
```julia
] add RandLinearAlgebra
```
in the Julia REPL. 

For the latest version, you can do the following. 

```julia
] add https://github.com/numlinalg/RandLinearAlgebra.jl.git
```

It is also possible to clone the repository into a local directory. In that case refer to 
Julia Pkg instructions:
[Using someone else's project](
    https://pkgdocs.julialang.org/v1/environments/#Using-someone-else's-project
)
